Featured Project: Global Goals Interactive

This interactive artwork was created in collaboration with Carolyn and highlights four UN Sustainable Development Goals through visuals and symbolic design: Zero Hunger, Reduced Inequalities (Jayesh), and Clean Water and Quality Education (Carolyn). Each quadrant of the canvas is color-coded and features unique graphics to reflect its theme, from stylized icons to animated lines and text.

Concepts Learned

  • Positioning and layout with coordinates in creative coding
  • Working with shapes like polygons, stars, circles, and gradients
  • Layering and z-order in graphical design
  • Typography and label design in visual storytelling
  • Embedding collaborative work in a personal website

Project 2: Climate Action Interactive

This interactive slideshow project explores the global goal of Climate Action. Created in collaboration with Meet, it walks viewers through the causes, effects, and solutions to climate change using animated graphics and educational slides. I contributed to the visual designs and explanations for how climate change affects us, and what solutions can help reduce its impact.

Concepts Learned

  • Creating interactive, multi-slide experiences using mouse events
  • Designing informative visuals and symbols (like thermometers, clouds, suns)
  • Using functions to organize repetitive drawing logic
  • Explaining real-world climate issues through visual code storytelling

My windMills() Function

This function is responsible for handling all energy gain based on windmill level. It checks whether the player has unlocked better windmills and adds energy accordingly. Here‚Äôs a simplified version of it:

def windMills(energyAdded, windMillLevel):
    if windMillLevel == 'Bronze':
        Energy.value += energyAdded
    elif windMillLevel == 'Silver' and Energy.value >= 1000:
        Energy.value += energyAdded
    elif windMillLevel == 'Gold' and Energy.value >= 2500:
        Energy.value += energyAdded

# Respond to user answers with key input

def onKeyPress(key):
    if app.questionNum == 1:
        if key == 'c':
            app.correct += 1
            correctLabel.value = 'Correct!'
            correctLabel.fill = 'green'
            correctLabel.visible = True
            correctLabel.toFront()
            app.questionNum += 1
            sleep(0.08)
        else:
            correctLabel.value = 'Incorrect, try again'
            correctLabel.fill = 'red'
            correctLabel.visible = True
            correctLabel.toFront()

Project 5: Zero Hunger Delivery Maze

In this game, the player drives a food truck to collect food from green stations and deliver it to purple houses, representing hungry children in need. This supports the UN goal of Zero Hunger. The maze structure and logic encourage exploration and accurate deliveries.

Concepts Learned

  • Function abstraction with arguments
  • Collision detection with hitsShape() and user input events
  • Game state management (splash, in-game, win)
  • Layering with groups (wallGroup)

Key Custom Function: placeWalls(x, y, width, height)

def placeWalls(x, y, width, height):
    if (width >= 0 or height >= 0) or (x+width > house.left and y+height > house.top):
        wall = Rect(x, y, width, height, fill='red')
        wallGroup.add(wall)
        wallGroup.toFront()
